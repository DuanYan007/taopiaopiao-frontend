# 桃票网前端开发文档 - 购票流程

## 1. 购票业务流程

从用户浏览演出到完成购票的完整流程如下：

### 1.1 完整流程图

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  浏览演出   │ -> │  选择场次   │ -> │  选择座位   │ -> │  确认订单   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
                                                                   │
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┘
│  查看订单   │ <- │  完成支付   │ <- │  提交订单   │ <- │  锁定座位   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

### 1.2 详细步骤

| 步骤 | 操作 | 涉及接口 |
|------|------|----------|
| 1 | 浏览演出列表 | `GET /client/events` |
| 2 | 查看演出详情 | `GET /client/events/{id}` |
| 3 | 选择场次 | `GET /client/events/{eventId}/sessions` |
| 4 | 获取座位布局 | `GET /client/seat-templates/{templateId}/layout` |
| 5 | 锁定座位 | `POST /api/seckill/lock` |
| 6 | 创建订单 | `POST /client/orders` |
| 7 | 支付订单 | `POST /client/orders/pay` |
| 8 | 查看订单 | `GET /client/orders/{orderNo}` |

### 2.4 秒杀/锁座接口 (seckill-service)

#### 2.4.1 锁定座位

**请求**
```
POST /api/seckill/lock
Content-Type: application/json
```

**请求体**
```json
{
  "sessionId": 101,
  "userId": 10001,
  "seatIds": ["1001", "1002"],
  "expireSeconds": 900
}
```

**参数说明**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| sessionId | Long | 是 | 场次ID |
| userId | Long | 是 | 用户ID |
| seatIds | String[] | 是 | 座位ID列表 |
| expireSeconds | Integer | 是 | 锁定时长(秒)，建议900(15分钟) |

**响应示例**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "success": true,
    "code": 0,
    "message": "锁座成功",
    "lockId": "lock_1234567890",
    "lockedSeats": ["1001", "1002"],
```

**错误码说明**
| code | 说明 | 处理建议 |
|------|------|----------|
| 0 | 成功 | 继续下单 |
| 1 | 座位不存在 | 刷新座位列表 |
| 2 | 座位已锁定/售出 | 提示用户选择其他座位 |
| 3 | 重复购票 | 提示用户已锁过该座位 |

## 3. 前端开发注意事项

### 3.1 座位选择逻辑

**座位状态展示**
```javascript
// 座位状态颜色映射
const seatStatusColors = {
  available: '#52c41a',   // 绿色 - 可选
  sold: '#d9d9d9',        // 灰色 - 已售
  locked: '#faad14',      // 橙色 - 已锁定
  unavailable: '#595959', // 深灰 - 不可用
  selected: '#1890ff'     // 蓝色 - 已选
};

// 座位点击处理
const handleSeatClick = (seat) => {
  // 已售/已锁定/不可用座位不可选
  if (['sold', 'locked', 'unavailable'].includes(seat.status)) {
    return;
  }

  // 切换选中状态
  if (selectedSeats.has(seat.id)) {
    selectedSeats.delete(seat.id);
  } else {
    selectedSeats.add(seat.id);
  }

  // 更新UI
  updateSeatUI();
};
```

### 3.2 锁座超时处理

**倒计时实现**
```javascript
// 锁座倒计时
let countdown = 900; // 15分钟
let timer = null;

const startCountdown = () => {
  timer = setInterval(() => {
    countdown--;

    // 更新倒计时显示
    updateTimerDisplay(formatTime(countdown));

    // 最后30秒提醒
    if (countdown <= 30 && countdown > 0) {
      showToast('座位即将过期，请尽快完成支付');
    }

    // 倒计时结束
    if (countdown <= 0) {
      clearInterval(timer);
      handleSeatExpire();
    }
  }, 1000);
};

const handleSeatExpire = async () => {
  // 释放座位
  await releaseSeats(sessionId, userId, Array.from(selectedSeats));
  // 跳转回选座页
  router.push('/select-seats');
};
```

### 3.3 订单状态轮询

**支付轮询**
```javascript
// 轮询订单状态
const pollOrderStatus = async (orderNo) => {
  const maxAttempts = 60; // 最多轮询5分钟(60次*5秒)
  let attempts = 0;

  const poll = async () => {
    attempts++;

    try {
      const result = await getOrderDetail(orderNo);

      switch (result.data.status) {
        case 1: // 已支付
          stopPolling();
          router.push('/order/success');
          break;
        case 2: // 已取消
        case 4: // 超时取消
          stopPolling();
          router.push('/order/cancelled');
          break;
        case 0: // 待支付，继续轮询
          if (attempts >= maxAttempts) {
            stopPolling();
            showToast('支付超时，请重新下单');
          }
          break;
      }
    } catch (error) {
      console.error('轮询失败', error);
    }

    if (attempts < maxAttempts && result.data.status === 0) {
      setTimeout(poll, 5000); // 5秒后继续
    }
  };

  poll();
};
```

### 3.4 错误处理

**统一错误处理**
```javascript
// API错误处理
const handleApiError = (error) => {
  const errorMap = {
    1001: '座位不存在',
    1002: '座位已被锁定或售出',
    1003: '您已选择过该座位',
    2001: '订单不存在',
    2002: '订单已支付',
    2003: '订单已取消'
  };

  const message = errorMap[error.code] || error.message || '操作失败，请重试';
  showToast(message);
};

// 请求拦截器
axios.interceptors.response.use(
  response => response,
  error => {
    if (error.response) {
      handleApiError(error.response.data);
    } else if (error.request) {
      showToast('网络连接失败，请检查网络');
    }
    return Promise.reject(error);
  }
);
```

## 5. 附录：完整购票流程示例

```javascript
// 完整购票流程
async function purchaseTicket(sessionId, seatIds) {
  try {
    // 1. 锁定座位
    const lockResult = await lockSeats({
      sessionId,
      userId: getCurrentUserId(),
      seatIds,
      expireSeconds: 900
    });

    if (!lockResult.data.success) {
      showToast('锁座失败：' + lockResult.data.message);
      return;
    }

    // 2. 开始倒计时
    startCountdown(900);

    // 3. 创建订单
    const orderResult = await createOrder({
      sessionId,
      eventId: getCurrentEventId(),
      seatIds
    });

    const orderNo = orderResult.data.orderNo;

    // 4. 跳转到支付页面
    router.push(`/pay/${orderNo}`);

    // 5. 等待支付...
    // (用户完成支付后)

    // 6. 轮询订单状态
    pollOrderStatus(orderNo);

  } catch (error) {
    console.error('购票失败', error);
    // 释放座位
    await releaseSeats(sessionId, getCurrentUserId(), seatIds);
  }
}
```
